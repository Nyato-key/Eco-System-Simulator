# Eco-System-Simulator

The Ecosystem Simulation is an artificial life system that models population dynamics and evolution. It simulates autonomous creatures navigating an environment, consuming resources, and reproducing with genetic inheritance.

Objective

1. Simulate an evolving ecosystem with autonomous agents
2. Implement genetic inheritance with mutation
3. Study population dynamics and trait evolution
4. Provide interactive visualization and data analysis

The simulation uses:
1. Agent-based modeling: Individual Grubs with traits (speed, sight, metabolism)
2. Genetic inheritance: Traits passed to offspring with configurable mutation
3. Spatial partitioning: Uniform grid system for efficient collision detection
4. Resource management: Food and water consumption with dynamic respawning
5. Reproduction system: Population control through crowding factors
6. Data collection: Real-time statistics and CSV export for analysis

Tools:
1. C++: The core programming language used to implement all simulation logic,
including organism behavior, genetic inheritance, resource management, and
evolutionary algorithms using object-oriented programming principles.
2. Raylib: A cross-platform C++ library used for rendering 2D graphics, handling user
input, managing the application window, and the visualization system.

Limitations
1. Simplified 2D environment
2. Basic genetic model
3. Homogeneous resource distribution
4. Performance constraints limit population size

Conclusion
The simulation successfully demonstrates evolutionary principles and population
dynamics. It provides insights into how genetic traits affect survival and reproduction,
while serving as an educational tool for understanding ecosystem modeling. The modular
design allows for future enhancements and more complex simulations.
